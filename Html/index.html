Difference between block and inline elements

A div will take the entire line horizontally

Meaning that a second block element will be pushed to the next Line

However an inline element will take as much space as needed

A block element following it will appear right after it

A block element inside of a block will go to the next line

What about an inline Element inside of a block?

Even if I add Multiple spans it will still be in the same line created by the p tag.
